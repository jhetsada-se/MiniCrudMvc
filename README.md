# Mini CRUD MVC (.NET 8 + EF Core + SQLite)

I built this project to practice ASP.NET Core MVC with Entity Framework Core 8 and SQLite.  
It’s a small task management app where I tried out Code-First Migrations and learned how controllers and views interact.

## Setup & Run
1. Open the solution in Visual Studio 2022.  
2. Open Package Manager Console -> run ```Update-Database```.  
3. Press F5 to start.  
4. You’ll see a basic Todo page where you can add, edit, or delete tasks.

## Highlights
- CRUD operations using EF Core Code-First  
- SQLite database (```AppData.db```) created automatically  
- Clean Bootstrap 5 UI
